# **App Name**: ResiMeals

## Core Features:

- User Authentication: User authentication: secure sign-in/sign-up for residents, directors, and administrators.
- Meal Schedule Display: Meal Schedule Display: a clear and concise view of the weekly meal plan.
- Role-Based Access: User role management: Residents can view the meal schedule. Directors and Administrators have more management capabilities.

## Style Guidelines:

- Main color: blue #084886
- Second color: white #FCFFFF
- Third color: red #A21E35
- Clear and readable font choices for easy information consumption.
- Simple, intuitive icons to represent different meal types and dietary options.
- Clean and organized layout for easy navigation.
- Subtle transitions and animations to enhance user experience.

## Original User Request:
Create a mobile application for Android and iOS to manage meal schedules in a university student residence. The application has four types of users: residents, directors, administrators, and a master user. The master user only creates the residences and their administrators. Administrators create resident and director users, and configure everything necessary for the residence to function. Back-end in Firebase, front-end in flutter.
